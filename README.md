# The Hunger Games - Hangman

A Hunger Games themed hangman game. Guess characters, places, and items from the series before the tribute meets their fate.

**[Play Now](https://puneet2019.github.io/hunger-games-hangman/)**

## How to Play

- Each round presents a hint related to The Hunger Games universe
- Guess letters using the on-screen keyboard or your physical keyboard
- You get 6 wrong guesses before it's game over
- Win to unlock a surprise

## Run Locally

```bash
make start   # starts a local server at http://localhost:8000
make stop    # stops the server
```
